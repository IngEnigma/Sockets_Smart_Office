Sistema de Iluminación y Temperatura para una Oficina

Ajuste Automático de Iluminación y Temperatura
El sistema regula la iluminación y la temperatura con base en distintos factores ambientales para garantizar un ambiente cómodo y eficiente.


Sensores Utilizados:

Temperatura interior y exterior → Para determinar la necesidad de calefacción o aire acondicionado.

Humedad → Para mantener un nivel adecuado de confort.

Luz → Para ajustar la iluminación y el control de persianas.

Calidad del aire → Para evaluar la necesidad de ventilación.



Actuadores Disponibles:

luces → Ajusta la intensidad de la iluminación artificial.

Servos ventanas → Abren o cierran las ventanas según las condiciones.

Servos para persianas o cortinas → Regulan la entrada de luz natural.

aire acondicionado → Enfría el ambiente cuando la temperatura es alta.

calefacción → Aumenta la temperatura cuando hace frío.



Ejemplo de Funcionamiento:
Temperatura agradable (ni frío ni calor):

Las ventanas y persianas permanecen abiertas para ventilación.

Mucho viento:

Se cierran las ventanas para evitar corrientes fuertes, pero las persianas siguen abiertas.

Mucha luz directa:

Se cierran las persianas parcialmente o completamente según la intensidad de la luz.

Frío:

Se cierran las ventanas y se enciende la calefacción.

Calor:

Se cierran las ventanas y se activa el aire acondicionado.
